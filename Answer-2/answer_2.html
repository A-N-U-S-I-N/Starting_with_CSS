BEM Naming Convention

  1. B stands for Block. For naming of block elements use small alphabets with parts using hyphen.
  2. E stands for Element. For naming of elements we use small alphabets using block element at first then using double underscore separation then the element naming.
  3. M stands for Modifier. For naming of modifiers we use small alphabets using element name first then the modifier name separated by two hyphens.

CSS Class Selector

Class selectors are written by having dot symbol(.) with the class name. After selecting the class write the property and its value within curly braces.
